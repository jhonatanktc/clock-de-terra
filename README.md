# clock de terra

Traditional clocks have "illogical" designs, such as the hour hand rotating twice a day, causing visual confusion between AM and PM. clock-de-terra breaks this habit by returning to a more intuitive, global perspective.

1. True 24-Hour Cycle
    - Bottom: 00:00
    - Right: 06:00
    - Top: 12:00
    - Left: 18:00
2. Global Synchronicity

    There are no minute or second hands. Instead, the clock features 24 hour hands, each representing one of the 24 time zones. This allows you to see the current rhythm of the entire planet at a single glance.
