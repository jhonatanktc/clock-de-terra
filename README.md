# clock de terra

[https://jhonatanktc.github.io/clock-de-terra/clock-de-terra.html](https://jhonatanktc.github.io/clock-de-terra/clock-de-terra.html)

傳統指針時鐘的上午和下午在視覺上完全相同，唯一的時針只能表示一個時區。若從北極上空俯瞰地球，地球是逆時針自轉，所以傳統指針時鐘的轉動方向也不對。所以 clock de terra 就是一個更合理的指針時鐘：一天只逆時針轉動一圈，且使用24個時針顯示所有時區的時間，而年、月、日、分和週改用數位時鐘顯示。
